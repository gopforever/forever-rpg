Getting Started

Contents

What is Forever RPG? edit

Forever RPG is a browser-based, massively-flavored RPG set in the world of Norrath, inspired by classic EverQuest. You assemble a party of adventurers, choose their race and class, and send them into progressively more dangerous zones to fight monsters, collect loot, earn achievements, and climb the global rankings.

The game runs entirely in your web browser with no installation required — just a local HTTP server. It uses 100% vanilla JavaScript with no external frameworks or dependencies.

System Requirements edit

  • A modern web browser (Chrome, Firefox, Edge, or Safari)
  • A local HTTP server (see Running the Game below)
  • No plugins, no accounts, no installation
Note: The game cannot be opened directly as a file:// URL — it must be served over http:// so that JavaScript modules load correctly.

Running the Game edit

Clone or download the repository, then start a local HTTP server from the repository root:

Using npx serve

If you have Node.js installed:

npx serve .

Then open http://localhost:3000 in your browser.

Using Python

If you have Python 3 installed:

python -m http.server 8080

Then open http://localhost:8080 in your browser.

Using VS Code Live Server

Install the Live Server extension, right-click index.html, and select Open with Live Server.

Character Creation edit

When you first load the game, the Character Creation overlay appears. Here you build your adventuring party before entering the world.

Party Size

You can assemble a party of up to 6 characters. Each character is created independently with their own race and class combination. A balanced party typically includes a tank, a healer, and several DPS or support characters.

Choosing a Race

Each race provides different base statistics (STR, DEX, AGI, STA, WIS, INT, CHA) and restricts which classes that character can be. For example:

  • Ogre — Highest STR/STA, but limited to Warrior, Shadow Knight, and Shaman
  • Erudite — Highest INT (107), excellent spellcasters
  • Human — Balanced stats with access to all 14 classes

See the Races article for the full table.

Choosing a Class

Classes define your character's role in combat and determine which abilities they learn. The 16 classes are grouped into four archetypes:

  • Melee — Warrior, Monk, Rogue, Berserker
  • Hybrid — Paladin, Shadow Knight, Ranger, Bard, Beastlord
  • Caster — Wizard, Magician, Necromancer, Enchanter
  • Priest — Cleric, Druid, Shaman

See the Classes article for the full guide.

Beginning the Adventure

Once you've created at least one character, click "Begin Adventure" to enter the world. All characters start in Qeynos — The City of Honor.

UI Overview edit

The game UI is divided into four main regions:

Top Bar

Runs across the top of the screen. Contains the game logo, the current zone name, the game clock, your wallet (gold / silver / copper), a link to the Rankings page, the Settings panel, and the Save button.

Left Sidebar

The left sidebar contains five panels:

  • Zone — Shows the minimap and zone information for your current location
  • Party Roster — Lists all characters in your party with their current HP/Mana
  • Kill Count — Tracks how many monsters you've slain in the current session
  • Who's Online — Shows other players' ghost characters in the world
  • Monster Log — A running log of recent enemy encounters

Center Panel

The center is the heart of the game:

  • City Panel (when in Qeynos) — Five tabs: Bank, Market, Guild, Players, Rankings
  • World Feed — Real-time activity from the game world
  • Combat Area — Enemy list, combat buttons (Stop/Sit/Pull/Camp), spell hotbar, combat log, zone chat, loot display, and the enemy encounter grid

Right Sidebar

The right sidebar shows your selected character's details:

  • Character Inspect — Name, race, class, level, and portrait
  • Stats — All attributes (STR, DEX, AGI, STA, WIS, INT, CHA) and derived stats
  • Equipment — Currently equipped gear across all slots
  • Abilities — Learned class abilities with cooldown status
  • Inventory — Unequipped items in your character's bag

Save & Load edit

Auto-Save

The game automatically saves your progress to your browser's localStorage at regular intervals. Your save persists between browser sessions as long as you use the same browser and don't clear site data.

Export / Import

Use the Settings panel (top bar) to export or import your save as a JSON file:

  • Export Save — Downloads your current save data as a .json file. Use this to back up your progress or transfer it to another machine.
  • Import Save — Upload a previously exported JSON file to restore a save.
  • Delete Save — Permanently removes your current save from localStorage.
  • Hard Reset — Completely resets the game to its initial state.

Settings

The Settings panel also exposes:

  • Auto-Loot Toggle — When enabled, loot from defeated enemies is automatically added to your inventory without requiring a click.
  • Combat Speed Slider — Adjusts the tick rate from 500 ms (fast) to 5000 ms (slow). See Combat Speed for details.
